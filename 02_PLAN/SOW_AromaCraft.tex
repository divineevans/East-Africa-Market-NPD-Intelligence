\section{1. Project Overview}\label{project-overview}

AromaCraft Ingredients Ltd.~requires a \textbf{Market \& NPD
Intelligence Report} for \textbf{East Africa (Kenya, Uganda, Tanzania,
Ethiopia)} to guide flavour pipeline development, pack \& price
architecture, and consumer engagement strategy.

This document defines scope, deliverables, data sources, KPIs, risks,
and timeline following the \textbf{APAPASA}
(Ask--Plan--Prepare--Analyse--Share--Act) framework.

\section{2. Objectives}\label{objectives}

\begin{itemize}
\tightlist
\item
  Quantify market size and growth (Volume \& Value, CAGR 2019-2024)
\item
  Identify top NPD launches \& flavour trends across beverage, bakery \&
  snack categories
\item
  Map key players \& trade/investment flows
\item
  Profile Gen Z vs Millennial consumers
\item
  Recommend flavour \& pack formats for 2026 innovation pipeline
\end{itemize}
